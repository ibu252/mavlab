Testing MAVLab Website Locally

Option 1: Use Vite Preview (Recommended)

Run this command in the terminal:

npm run preview

This will serve the dist folder correctly with all paths working.

Option 2: Use Python HTTP Server

Navigate to the dist folder and run:

cd dist
python3 -m http.server 8000

Then open: http://localhost:8000

Option 3: Use Node.js http-server

Install and run:

npx http-server dist -p 8000

Then open: http://localhost:8000

Important Notes:

  • For local testing: Use npm run build (base: '/')
  • For GitHub Pages: Use npm run build:gh-pages (base: '/mavlab/')
  • Don't use Live Server extension with GitHub Pages build - paths won't match
